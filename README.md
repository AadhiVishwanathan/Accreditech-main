# Accreditech

Accreditech is a web platform designed to simplify and digitize the institute accreditation and approval process.  
It allows institutions to submit applications, upload required documents, and manage the approval workflow efficiently.

---

## 🚀 Features

- 🔐 **User Management**
  - Role-based access (Institutions, Evaluators, Admins)
  - Secure login and authentication
- 📝 **Application Submission**
  - Multi-step accreditation forms
  - Document upload and validation
- 📊 **Dashboard & Tracking**
  - Real-time status updates
  - Progress tracking for applications
- 📂 **Document Management**
  - Upload, review, and manage accreditation documents
- 📜 **Reports & Analytics**
  - Generate reports for administrators and evaluators

---

## 🛠️ Tech Stack

- **Frontend:** React (Vite + TypeScript), Tailwind CSS, shadcn-ui  
- **Backend:** Node.js (Express.js)  
- **Database:** MongoDB / Supabase  
- **Authentication:** JWT / Supabase Auth  
- **Other Tools:** Git, GitHub, Lovable (optional for prototyping)

---

## 📦 Installation

Follow these steps to set up Accreditech locally:

```bash
# Clone the repository
git clone https://github.com/yogabalajir24/accreditech.git

# Navigate into the project directory
cd accreditech

# Install dependencies
npm install

# Start the development server
npm run dev
